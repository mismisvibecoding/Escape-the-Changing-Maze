## 🎮 Escape the Changing Maze

### Description
A fast-paced browser arcade game where you're trapped inside a procedurally generated maze that changes every level. Navigate through darkness, dodge intelligent monsters, collect coins for massive combos, grab power-ups, and reach the exit portal before time runs out. Every run is different — the maze, enemy placement, and power-up locations are randomized each time.

---

### 🎯 Goal
Escape each maze by reaching the **purple exit portal** before the timer hits zero, while avoiding monsters and collecting as many coins as possible. Survive as many levels as you can and chase the highest score on the leaderboard.

---

### 🕹️ How to Play

| Control                       | Action                                           |
|-------------------------------|--------------------------------------------------|
| **W A S D** or **Arrow Keys** | Move through the maze                            |
| **Spacebar**                  | Dash (burst of speed, can pass through enemies)  |
| **E** or **Shift**            | Emergency Blast (pushes all nearby enemies away) |

### ⚡ Power-ups
- **⚡ Speed** — Move faster for 6 seconds
- **❄ Freeze** — Stops all enemies for 5 seconds
- **🛡 Shield** — Absorbs one monster hit
- **⏰ Time** — Adds +12 seconds to the timer
- **🧲 Magnet** — Pulls nearby coins to you for 6 seconds

### 👾 Enemy Types
- **Red (Chaser)** — Hunts you directly
- **Orange (Patrol)** — Wanders randomly, chases if you get close
- **Purple (Teleporter)** — Chases you and teleports nearby periodically

### 💡 Tips
- **Chain coin pickups** within 2.5s to build combos (up to **10× score multiplier**)
- **Dash through** an enemy for +50 bonus points
- **Near-miss** bonus: pass close to a monster without touching for +25 points
- Use the **minimap** (bottom-right) to track explored areas and the exit
- Save your **Blast** for emergencies when cornered by multiple enemies
- Difficulty increases each level: more enemies, faster speed, bigger maze
